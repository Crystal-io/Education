| Метрика | Причины падения | Способы улучшения |
|--------|----------------|-------------------|
| CPM | высокая конкуренция, слабый таргетинг | оптимизация аудиторий, смена GEO, креативы |
| CTR | плохие креативы, нерелевантная аудитория | A/B тест баннеров, персонализация, офферы |
| CPC | высокая конкуренция, низкий CTR | улучшение CTR, оптимизация bidding стратегии |
| Click-to-Visit Rate | редирект лаги, трекинг ошибки | оптимизация трекера, CDN, сокращение редиректов |
| Bounce Rate | медленный сайт, нерелевантный лендинг | ускорение загрузки, улучшение UX/UI, релевантные лендинги |
| CR (Click→Reg) | сложная форма регистрации, слабый оффер | упрощение регистрации (1-click, соцсети), улучшение value proposition |
| Cost per Registration | дорогой трафик, низкий CR | оптимизация каналов, улучшение CR |
| CR (Reg→KYC) | сложный KYC, UX friction | упрощение KYC, авто-верификация, better UX |
| CR (KYC→FTD) | проблемы с платежами, недоверие | добавить PSP, локальные методы, trust badges |
| CR (Reg→FTD) | плохой onboarding, слабый оффер | welcome бонусы, onboarding flow, email/push |
| CPA | дорогой трафик, низкий CR | улучшение CR, оптимизация каналов |
| FTD Rate | некачественный трафик | фильтрация источников, антифрод |
| Time to FTD | сложный cashier, низкая мотивация | упрощение депозита, instant payments, бонусы |
| Avg First Deposit | слабый UX, низкий trust | минимальный депозит, upsell бонусами |
| Deposit Frequency | плохое удержание | CRM кампании, бонусы, VIP программы |
| Bet Amount | низкий engagement | gamification, турниры, рекомендации |
| Hold (%) | RTP игр, поведение игроков | настройка game mix, управление RTP |
| GGR | низкая активность или hold | рост трафика, улучшение retention |
| NGR | высокие бонусы/комиссии | оптимизация бонусной политики |
| Bonus Cost Ratio | слишком агрессивные бонусы | tightening бонусных условий |
| ARPU | низкая активность | CRM, персонализация |
| ARPPU | слабые high-value игроки | VIP сегментация, персональные офферы |
| LTV | плохой retention | улучшение retention, CRM |
| Payback Period | высокий CPA, низкий ARPU | снижение CPA, рост ARPU |
| ROAS | неэффективные каналы | перераспределение бюджета |
| ROI | низкий доход или высокий spend | оптимизация всей воронки |
| Withdrawal Amount | игроки быстро выводят деньги | улучшение retention, бонусы |
| Net Cash Flow | много выводов | удержание игроков |
| Retention (D1/D7/D30) | плохой продукт, слабый onboarding | улучшение UX, CRM, геймификация |
| Churn Rate | низкая вовлеченность | CRM, персонализация |
| Session Frequency | нет причин возвращаться | пуши, акции, турниры |
| Avg Session Duration | скучный продукт | UX, контент, UX |
| Bet Frequency | низкий engagement | быстрые игры, autoplay |
| Active Days | слабый retention | CRM, акции, VIP |
