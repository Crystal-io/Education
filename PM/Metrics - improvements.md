| Метрика | Причины падения | Способы улучшения | Примеры улучшения |
|--------|----------------|-------------------|-------------------|
| CPM | высокая конкуренция, широкий таргетинг | оптимизация таргетинга, GEO split | исключить дорогие GEO, сузить аудиторию (например, только high-intent users) |
| CTR | слабые креативы, нерелевантный оффер | A/B тесты, персонализация | протестировать 5 баннеров с разными офферами (bonus vs no-deposit) |
| CPC | низкий CTR, плохой quality score | улучшение CTR, оптимизация кампаний | повысить CTR → снизится CPC в Google Ads |
| Click-to-Visit Rate | редиректы, медленный трекер | оптимизация редиректов | убрать лишние redirect hops, использовать server-side tracking |
| Bounce Rate | медленный лендинг, mismatch ожиданий | улучшение UX, скорость загрузки | ускорить страницу до <2 сек, синхронизировать оффер в рекламе и лендинге |
| CR (Click→Reg) | длинная регистрация, low trust | упростить регистрацию | сделать 1-step регистрацию + social login |
| Cost per Registration | дорогой трафик, низкий CR | оптимизация каналов + CR | отключить неэффективные кампании с высоким CPC |
| CR (Reg→KYC) | сложный KYC, долгий процесс | упрощение KYC flow | внедрить auto-KYC (Sumsub / Onfido) |
| CR (KYC→FTD) | проблемы с оплатой, нет локальных методов | добавить PSP, локальные методы | добавить Apple Pay / Google Pay / локальные APM |
| CR (Reg→FTD) | слабый onboarding | улучшить onboarding | welcome flow: email + push + бонус на депозит |
| CPA | высокий CPC, низкий CR | оптимизация funnel | улучшить CR (Reg→FTD) вместо увеличения бюджета |
| FTD Rate | некачественный трафик | фильтрация каналов | отключить affiliate с низким FTD |
| Time to FTD | сложный депозит | упростить cashier | one-click deposit, autofill card |
| Avg First Deposit | низкое доверие | стимулы к увеличению депозита | бонус 100% при депозите от $50 |
| Deposit Frequency | нет причин вернуться | CRM кампании | weekly reload бонусы |
| Bet Amount | низкий engagement | рекомендации, UX | показывать “popular games” и персональные рекомендации |
| Hold (%) | высокий RTP игр | управление game mix | увеличить долю игр с RTP 94% vs 98% |
| GGR | низкий bet volume | рост активности | турниры и leaderboard |
| NGR | высокие бонусы | оптимизация бонусов | снизить wagering multiplier |
| Bonus Cost Ratio | агрессивные бонусы | пересмотр бонусной политики | убрать бездепозитные бонусы |
| ARPU | низкая активность | стимулирование активности | CRM кампании, персональные офферы |
| ARPPU | мало high-value игроков | VIP программы | выделить VIP сегмент с менеджером |
| LTV | слабый retention | улучшение retention | loyalty program + миссии |
| Payback Period | высокий CPA | рост ARPU | увеличить retention → растет LTV |
| ROAS | неэффективные каналы | перераспределение бюджета | увеличить spend в каналах с ROAS >2 |
| ROI | высокий cost / низкий revenue | оптимизация экономики | снизить бонусы + улучшить retention |
| Withdrawal Amount | быстрые выводы | удержание игроков | бонус на повторный депозит после вывода |
| Net Cash Flow | высокий withdrawal | увеличение депозитов | CRM на возврат игроков |
| Retention (D1/D7/D30) | плохой onboarding | улучшение first experience | onboarding миссии + tutorial |
| Churn Rate | отсутствие ценности | персонализация | personalized offers |
| Session Frequency | нет триггеров возврата | CRM | push “you have free spins” |
| Avg Session Duration | слабый UX | улучшение контента | добавить live games / турниры |
| Bet Frequency | низкая динамика игр | ускорение gameplay | autoplay / quick bet |
| Active Days | нет привычки | регулярные стимулы | daily rewards / streaks |
